# ShyftLabs Assignment 

1. Ensure that the latest version of Python is installed
    - Windows
      1. Download the latest version from the [Python Website](https://www.python.org/downloads/)
      2. Ensure that the "Add to path" checkbox is selected during install
    - Linux (Ubuntu/Debian)
      1. Open terminal
      2. `sudo apt update`
      3. `sudo apt install python3`
2. Download/clone project and open it in terminal/cmd
3. Run the following command
     - Linux: `python3 server.py 1234`
     - Windows: `python server.py 1234`
4. In your browser navigate to `localhost:1234`
5. Terminate the server with `^c` in terminal/cmd
